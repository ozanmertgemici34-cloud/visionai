# Repository Notice

## What This Is

This is the public showcase folder for **F.R.I.D.A.Y.** — a personal Windows desktop AI assistant. It contains documentation, architecture notes, a development roadmap, and simplified preview code intended to communicate the system design to outside readers.

This is **not** the source code repository.

---

## What Is Not Here

The following are intentionally excluded:

- Core application logic and pipeline code
- LLM prompting framework and system prompts
- Tool implementations (desktop automation, web access, file operations)
- Memory engine internals and retrieval logic
- Routing and decision logic between local and cloud models
- API configuration and credentials
- User-specific memory data
- Session logs and conversation history

All code samples in the `preview/` folder are simplified pseudocode written to illustrate design concepts. They are incomplete by design and do not reflect the actual implementation.

---

## What Is Accurate

The following **are** faithfully represented:

- Overall system architecture and component relationships
- Capabilities and their current implementation status (see `ROADMAP.md`)
- Design principles and privacy stance
- The general behavior of the memory system (structure, categories, retrieval concept)
- The general structure of the voice pipeline
